En esta carpeta se encuentran los ejercicios HTML. En ellos primero he hecho algunas pruebas con los conocimientos
que hemos ido adquiriendo a lo largo de las sesiones y luego he hecho un ejercicio final intentando hacer un 
ejercicio/proyecto algo más serio.

El ejercicio del Cv es un poco más simple y ha sido utilizado para las mediaqueries y para practicar
algunos conocimientos.

El Html_Ejercicio1 ha sido dividido en 3 partes:

	-Html_Ejercicio1: Para ver un poco las estructuras que vimos en css y aplicar un poco de todo lo que vimos.

	-Html_Ejercicio1_CSS: Para practicar tanto el Css como algunos aspectos de posicionamiento, aunque la 
			      página quedara descompensada y no muy bonita.

	-Html_Ejercicio1_Final: Un ejercicio algo más serio donde se pretendia simular una página oficial de la
				FBM dentro de los conocimientos adquiridos.
